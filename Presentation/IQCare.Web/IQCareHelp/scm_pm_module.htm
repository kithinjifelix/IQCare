IQCare Supply Chain Management

  

The Supply Chain Management module is tightly integrated with the IQCare's clinical web-based user interface. The pharmacy form and patient cost display in IQCare is driven by the supply change set up  and item configuration done in IQCare Management.

 

Any type of item can be configured in IQCare's supply chain management module. To learn more about the features, see the PM/SCM section of this user guide.
