Add an Entry in a Custom List

  

Adding entries to a custom list has several for

•Unrestricted lists

•Restricted lists

•List with and without codes

 

1.From the navigation bar, select Customize Lists

 

2.Select the list you would like to modify by clicking anywhere in the row with the list information. The list entries will display in a table in priority order.

3.Click the Add button

4.Fill in the list entry name, possibly a code and the priority (note that the entry status is assumed to be “active”). See section on list priorities for more information on how the priorities work.

•For unrestricted lists, when adding an item, you can add it with the priority you wish.

•For restricted lists, when adding an item, it will add to the end of the list restricted portion of the list and within the priority specified in the latter section

 

5.Click Save

6.The new entry will display in the location based on the priority you gave it
