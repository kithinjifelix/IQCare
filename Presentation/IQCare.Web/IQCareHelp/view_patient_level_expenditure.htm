View Patient Costs/Debit Note

  

Patient costs are added up on a ground-up basis. Overhead and services are allocated per visit based on the budget and number of patients that visit the clinic per month. (See Cost Allocations). Pharmacy items are purchased by the number of pills/items dispensed. Laboratory costs are by lab test.

 

Patient costs are available to see via the Patient's home page in the Patient Cost tab. All patient transactions that have a cost configured will display by debit note date.

 

Generate a Debit Note and See Cost History

1.To add a debit note (bill) for a patient, first, click on Patient Reports -> Debit Note.

 

 

 

2.A listing of all of the services, drugs and labs that the patient has used or purchased since the last debit note was generated displays. You can change the dates to reduce the number of line items and narrow the date range.

3.Once you have the dates you want, click submit for the debit note to be created.The debit note displays as a Crystal Report and can be printed.

 

 

4.To see the history of prior debit notes, go back to the Patient Home Page/Patient Costs tab. To see the debit note line items once again, click on the Bill # link of the selected date. If you want to print the debit note click on the link for the selected date. The debit note displays.
