Query Builder Reports

  

IQCare provides a way to pre-define reports in IQCare Management and run the reports in IQCare. Queries saved using the Query Builder are accessible under the Report navigation.

 

Create a query in IQCare Management

1.In IQCare Management, create and preview your report.

2.Save the report under a category and name the report. In this example, the category is Patient Visit Reports and the report name is Vital Signs.

 

 

3.Save the report under a category and name the report. In this example, the category is Patient Visit Reports and the report name is Vital Signs.

 

To run a report in IQCare

1.Select the report category

2.Then select the report name

 

3.Click Submit to run the report

4.IQCare will open Excel and the report results will display.
